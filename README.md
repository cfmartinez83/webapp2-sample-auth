# webapp2-sample-auth
Sample authentication using webapp2_extras.auth

How to run
- dev_appserver.py webapp2-sample-auth

Dev Url
- http://localhost:8080/
 
Admin Url
- http://localhost:8000/

Routes
- /login/
- /logout/
- /create/
- /secure/
